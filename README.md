# Bevy Climbing System

## What is it?
This is a little project I'll be working on for a couple of days - a Dying Light-style climbing system made entirely in Bevy game engine!
I made a similar project a year ago, but that was in Unity engine, and the code was unoptimized. My goal with this current project is to create a flexible system which will analyze the environment in real time (no manual marking required!) and let you climb places you could realistically grab onto.

## Features
### What I'd like this system to have (and what I've already implemented) is the following:

~~Basic movement (running and jumping)~~  
~~Grabbing onto ledges~~  
~~Dropping off ledges~~  
Jump off a ledge  
~~Climbing and shimmy'ing on a ledge~~  
Ability to shimmy around or in a corner (e.g. switching sides)  
~~Mantle onto an object~~  
Animated Arms Model  
Smaller tweaks  

## Instructions
### Here's how to use the current features:
W/A/S/D - Move (on surface or while climbing)  
Mouse - Look around  
Space - Jump (keep holding mid air to grab onto an available ledge)  
Shift - Drop from a ledge
